# Аудит приложения Divergents Super App — UX и дизайн

Документ — это итог аудита, проведённого старшей командой по всем экранам приложения
(LMS, Сообщество, Карьера, Карта, AI, профиль, авторизация, онбординг, навигация),
а также по дизайн-системе (`src/theme`, `src/components`).

Для каждой находки указан приоритет и статус:
- **СДЕЛАНО** — исправлено в этом проходе.
- **TODO** — остаётся к доработке.

Приоритеты: **HIGH** (критично для консистентности/юзабилити), **MED** (заметная
шероховатость), **LOW** (полировка).
