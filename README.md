# Divergents Super App

Мобильное приложение экосистемы **Divergents** — платформы личного развития на
основе психометрических данных (Gallup CliftonStrengths34, Gardner, MBTI).
Построено на **React Native + Expo (TypeScript)** в эстетике Apple Human
Interface Guidelines (iOS 17–18), по дизайну из Claude Design.

## Что внутри

Приложение разделено на 6 модулей, доступных через нижний таб-бар:

| Таб | Модуль | Статус |
|---|---|---|
| Обучение | LMS — курсы, уроки, видеоплеер | **Глубокая реализация** с реальной логикой прогресса |
| AI | Персональный AI-ассистент (чат) | Интерактивный чат с профиль-зависимыми ответами |
| Сообщество | 21-дневные челленджи, поездки, участники | **Глубокая реализация** с живым подсчётом очков |
| Карьера | Вакансии по совпадению с психотипом | Экран со списком и Best Match |
| Профиль | Живая карта личности, отчёты, таланты | Полный дашборд |

## Реальная логика (не просто верстка)

**LMS (`src/state/CourseContext.tsx`)**
- Прогресс курса считается из завершённых уроков.
- Карточка «Продолжить» сама находит текущий урок.
- Уроки разблокируются последовательно (done → current → available → locked).
- Кнопка «Завершить урок» в видеоплеере отмечает урок и обновляет прогресс
  во всех экранах (главная, каталог, карточка курса, профиль).

**Сообщество (`src/state/ChallengeContext.tsx`)**
- Челлендж «21 день без сахара» = 3 задачи: 10 000 шагов, день без сахара,
  10 страниц книги.
- Метрические задачи (шаги, страницы) можно увеличивать/уменьшать степпером.
- **Превышение нормы = бонусные очки**, которые идут команде (механика из ТЗ):
  напр. 13 240 / 10 000 шагов → +базовые + бонус.
- Очки за день суммируются и **в реальном времени меняют место в лидерборде
  команды** — логика подсчёта в чистых функциях `taskPoints`/`taskBonus`.

## Стек

- **Expo** SDK 52 / React Native 0.76 / React 18
- **React Navigation 7** — bottom tabs + native stacks
- **react-native-svg** — SF-Symbols-подобные иконки (нарисованы оригинально)
- **expo-blur** — материальный таб-бар
- TypeScript (strict)

## Структура

```
src/
  theme/        # дизайн-токены (цвета, типографика iOS Dynamic Type)
  components/   # SFIcon, ui-атомы, headers, Screen, ChallengeTaskRow
  navigation/   # таб-бар + стеки модулей
  data/         # моковые данные (курсы, челлендж, профиль, карьера, AI)
  state/        # контексты: прогресс курсов, состояние челленджа
  screens/      # экраны по модулям (lms / community / ai / career / profile)
```

## Запуск

```bash
npm install
npm start          # затем нажмите i (iOS) / a (Android), или сканируйте QR в Expo Go
```

> Данные сейчас **моковые** (захардкожены в `src/data`). Бэкенд (Supabase/
