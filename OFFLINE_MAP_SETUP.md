# Офлайн-карта (MapLibre) — настройка и запуск

Офлайн-карта работает на движке **MapLibre**, который требует нативный модуль.
Поэтому она запускается **не в Expo Go**, а в собственном dev-build приложении.

## 1. Стиль карты
По умолчанию используется демо-стиль MapLibre (`https://demotiles.maplibre.org/style.json`) —
он бесплатный, но низкодетальный (уровень стран). Для **уличных карт и нормального офлайна**
получите бесплатный ключ MapTiler и задайте переменную окружения:

```
EXPO_PUBLIC_MAP_STYLE_URL=https://api.maptiler.com/maps/streets-v2/style.json?key=ВАШ_КЛЮЧ
```

(положите в `.env` или экспортируйте перед сборкой/запуском).

## 2. Сборка dev-build (один раз)
```
npm install -g eas-cli
eas login
eas build --profile development --platform android   # или ios
```
Установите получившийся APK/приложение на телефон.

## 3. Запуск
```
npx expo start --dev-client -c
```
Откройте приложение (dev-build), не Expo Go.

## 4. Как пользоваться
Вкладка **Карта** → кнопка со стрелкой вниз (⤓) справа → экран **«Офлайн-карта»**.
Там: кнопка скачивания → выбор размера области (Район/Город/Регион) → загрузка с прогрессом.
Скачанные области видны в списке (можно удалить). В скачанных областях карта работает без интернета.

## Заметки
- Объём загрузки зависит от размера области и диапазона зума (10–16).
- Основная интерактивная карта осталась на react-native-maps (маршруты, поиск и т.д.).
  Офлайн — отдельный экран на MapLibre. При желании можно полностью перевести карту на MapLibre.
- Не качайте слишком большие области с публичных бесплатных серверов — соблюдайте лимиты провайдера.
